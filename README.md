
# Flower ML Classification

This project uses a **flower dataset** to classify species using Logistic Regression.  

## Features
- Length (cm)
- Width (cm)
- Size (cm)

## Target
- Species (Rose, Tulip, Lily)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run Python script: `python flower_classification.py`
3. Check predictions in `predictions.csv`
